Настройки CloudPosBridge

Логи

Логи программы хранятся в следующей папке:

Очищать логи при запуске Если включено, логи будут очищаться при каждом запуске службы
От 0 (минимум) до 5 (максимум). Чем выше уровень, тем больше информации записывается в логи
Номер COM-порта, к которому подключена касса
Имя кассира, которое будет использоваться при печати чеков
IP адрес контрольно-кассовой техники
Порт для подключения к контрольно-кассовой технике
IP адрес сервера контрольно-кассовой техники (если используется)
Включить эмуляцию Если включено, служба будет работать в режиме эмуляции без реального взаимодействия с ККТ
Укажите разрешенный origin для WebSocket соединений (например, http://localhost:3000). Оставьте пустым, чтобы разрешить все origins.
